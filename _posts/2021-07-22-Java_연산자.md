# 연산자

|연산 우선순위 |연산자 이름 |종류 |
|---|---|---|---|
|높음|**산술**|+, -, *, /, %|
||**부호**|+, -|
||**문자열**|+(문자열 연결)|
||**대입**|=, +=, -=, *=, /=, %=, &=, ^=, \|=, <<=, >>=, >>>= |
||**증감**|++, --|
||**비교**|==, !=, >, <, >=, <=, instanceof|
||**논리**|!, &, \|, &&, \|\||
||**조건**|(조건식)?A:B|
||**비트**|~, &, |, ^|
|낮음|**쉬프트**|>>, <<, >>>|


# NaN 과 Infinity 연산

try-catch 문을 사용하여 예외처리 한다.

```java
try{
    int z = x % y;
    System.out.println("z: " + z);
}catch(ArithmeticException e){
    System.oun.println("ArithmeticException");
}
```
